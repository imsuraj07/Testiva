# Testiva - Online Examination and Study Management Portal

## Description
Testiva is a web-based online examination portal that enables students to take exams, track performance, and access study resources. Administrators can manage exams, questions, users, and results efficiently.

## Features
- User Authentication & Authorization
- Exam Management
- Question Management
- Result Generation
- Student Performance Tracking
- REST API Integration

## Technologies Used
- Java
- Spring Boot
- React.js
- MySQL
- Spring Data JPA
- REST APIs
- Git & GitHub
